## Overview

SLIMEOUT is a lightweight one-button game built in Unity. Players travel through an unlimited maze, navigating obstacles and testing reflexes with simple controls.

## Key Features

- One-button gameplay loop
- Endless maze-style challenge
- Fast decision-making and reflex-based movement
- Built as an interactive game development project

## Technical Implementation

- Developed using **Unity**
- Programmed gameplay logic in **C#**
- Designed lightweight mechanics focused on simple input and replayability

## What I Learned

This project helped me practice interactive development, gameplay logic, player feedback, and building small systems that feel responsive.
