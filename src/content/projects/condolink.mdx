## Overview

CondoLink is my capstone project: a responsive web application for condominium residents, staff, guards, and administrators. The system centralizes building operations such as announcements, amenity booking, maintenance requests, reports, and entry logs.

## My Role

I worked as the full-stack developer and project lead, managing planning, feature development, testing, debugging, and delivery.

## Key Features

- Resident, staff, guard, and administrator dashboards
- Announcements and communication workflows
- Amenity booking and request tracking
- Maintenance request management
- Entry logs and reports
- Role-based access and user-specific features

## Technical Implementation

- Built front-end dashboards using **React**, **TypeScript**, and **Tailwind CSS**
- Developed PHP back-end logic for system workflows
- Connected MySQL tables for dynamic features and user-specific data
- Organized testing and debugging around real user flows

## What I Learned

This project strengthened my full-stack development skills, especially in role-based systems, database-backed workflows, project planning, debugging, and translating real operational needs into usable software.
