## Overview

This portfolio presents my profile as a BSIT candidate, IT support trainee, and junior web developer. It includes a hero section, project highlights, skills, experience, a resume download, and contact links.

## Key Features

- Responsive single-page layout for desktop and mobile
- Project cards for CondoLink and SLIMEOUT
- Contact section with email, LinkedIn, and GitHub links
- Resume download and quick links
- Built with Vite, React, and Tailwind CSS

## What I Learned

This project helped me practice building a polished personal website, organizing professional information, and deploying a static frontend to GitHub Pages.
